# react-native-csv

react-native-csv is the fastest CSV (or delimited text) parser for React Native.

## Installation

```sh
npm install react-native-csv
```

## Usage

```js
import Csv from "react-native-csv";

// ...

const result = await Csv.multiply(3, 7);
```
